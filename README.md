## Project Structure

This repository is organized into several folders to separate raw data, analysis scripts, and generated outputs.

## input_files
This folder contains the data used in the analysis scripts. These files include the datasets required to run the R scripts and reproduce the figures and visualizations in this project.

## output_files
This folder contains graphs and visualizations generated by the R scripts. These files are produced during the analysis and include plots used in the data report and presentation.

## R_scripts
This folder contains the R Markdown (.Rmd) files used for analysis and visualization. These scripts include:

- The data analysis report, which performs the exploratory data analysis and creates the figures.
- The Shiny application, which generates an interactive map for exploring shark movement and environmental conditions.
